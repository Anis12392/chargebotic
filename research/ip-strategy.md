# IP Protection Strategy — Chargebotic

*Date: Feb 25, 2026 | Budget: $0 | Location: San Francisco*

## Situation

2 cofondateurs, pas d'argent, pas encore incorporé. Besoin de protéger le concept de station de recharge murale universelle pour robots (wall-mounted vertical rail + bras robotique + induction + retrofit kit).

## Prior Art — Ce qui existe déjà

- **Ford** : Brevet rail-based robotic EV charger (Oct 2024) — pour voitures, pas robots
- **WiTricity** : IP dominante sur magnetic resonance — licensing large
- **InductEV** : 116+ brevets worldwide — recharge induction pour véhicules
- **Tesla** : Brevets wireless EV charging (2024) — alignment autonome

**Le gap** : AUCUN brevet identifié sur rail-mounted arm pour recharge de **robots** (vs. EV). La combinaison wall-mount + vertical rail + multi-type robot + retrofit est novel.

## Stratégie : Sniper Approach

### Breveter (hardware visible = reverse-engineerable)
- Wall-mount station design
- Rail vertical + mécanisme du bras
- Retrofit kit (receiver attaché au robot)
- Système d'alignement vision/IR

### Trade Secret (software = invisible)
- Algorithme de calibration bras-robot
- Fleet management architecture
- Données d'efficacité de charge
- Modèles de pricing
- Training data ML

5. Toutes les variantes (ceiling mount, corner mount, double-rail)
6. Le workflow complet : robot approach → detection → alignment → docking → charging
